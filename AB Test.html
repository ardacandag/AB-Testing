About Dataset

Data Source: https://www.kaggle.com/datasets/amirmotefaker/ab-testing-dataset

Below are all the features in the dataset:

  • 'Campaign Name': The name of the campaign
  • 'Date': Date of the record
  • 'Spend': Amount spent on the campaign in dollars
  • '# of Impressions': Number of impressions the ad crossed through the campaign
  • 'Reach': The number of unique impressions received in the ad
  • '# of Website Clicks': Number of website clicks received through the ads
  • '# of Searches': Number of users who performed searches on the website
  • '# of View Content': Number of users who viewed content and products on the website
  • '# of Add to Cart': Number of users who added products to the cart
  • '# of Purchase': Number of purchases

Two campaigns were performed by the company:

  • Control Campaign
  • Test Campaign

# Features and their indices for the x-axis
features = ['Spend [USD]', '# of Impressions', 'Reach', '# of Website Clicks', '# of Searches', '# of View Content', '# of Add to Cart', '# of Purchase']
len_features = np.arange(len(features))

# Mean values for each feature in control and test groups
control_means = df_control[features].mean()
test_means = df_test[features].mean()

# Bar width
bar_width = 0.35

# Create bar plots
plt.figure(figsize=(10, 6))
bar_control = plt.bar(len_features - bar_width/2, control_means, bar_width, label='Control')
bar_test = plt.bar(len_features + bar_width/2, test_means, bar_width, label='Test')

# Set x-axis labels
plt.xticks(len_features, features, rotation='vertical')

# Set plot labels and title
plt.xlabel('Features')
plt.ylabel('Mean Values')
plt.title('Comparison of Features between Control and Test Groups')
plt.legend()

# Add labels on top of the bars
for bar_control, bar_test in zip(bar_control, bar_test):
    height_control = bar_control.get_height()
    height_test = bar_test.get_height()
    plt.text(bar_control.get_x() + bar_control.get_width()/2 - 0.05, height_control + 0.05, f'{height_control:.1f}', ha='center', fontsize='small')
    plt.text(bar_test.get_x() + bar_test.get_width()/2 + 0.05, height_test + 0.05, f'{height_test:.1f}', ha='center', fontsize='small')

# Show the plot
plt.tight_layout()
plt.show()

# Perform t-test for'Spend [USD]'
control_spend = df_control['Spend [USD]']
test_spend = df_test['Spend [USD]']

t_stat, p_value = ttest_ind(control_spend, test_spend)

# Check if the difference is statistically significant
if p_value < 0.05:
    print("""The difference in 'Spend [USD]' between control and test groups is statistically significant.
             There is sufficient statistical evidence to conclude that the two samples have different means.""")
    print(f"T-stat: {t_stat: .4f}, P-value: {p_value: .4f}")
else:
    print("The difference is not statistically significant.")
    print(f"T-stat: {t_stat: .4f}, P-value: {p_value: .4f}")
